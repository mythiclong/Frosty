# Minecraft 26.2 Client For Skyblock

编程初学，这个客户端大概是在 **2025 年 8 月** 开始制作的。原本打算自己做一些功能改善自己的 **1.21.5** 游玩体验，后面逐渐完善了很多功能，到 **9月22号** 基本上完成到初次发布的程度

*"这是一个半死项目QWQ，随缘更新"*

**注意说明：**

* 本客户端因为个人水平原因和最初的开发原因，大量使用 **AI** 与参考了 **raven-bs** 客户端的框架，以及 **Meteor** 在高版本的方法。因此同样采用 **GNU General Public License v3.0** 开源协议
* 版本更新暂定在 **Hypixel** 每次强制更新时的最新版本，因为精力和能力有限，不会多版本同时适配
* 代码可能存在大量冗余和拖累进程的地方，介意可以不使用
